# AI Strategist - Diagnóstico de Integración de IA

Esta aplicación es una herramienta técnica y económica diseñada para evaluar la integración de Inteligencia Artificial Generativa en servicios de atención al cliente.

## 🚀 Objetivo del Proyecto
Facilitar la toma de decisiones estratégicas mediante el cálculo de ahorros operativos, la comparación de arquitecturas (API vs Cloud) y la gestión de riesgos (RAID).

## 🛠️ Funcionalidades
- **Calculadora Operativa:** Estimación de consumo de tokens y ahorro en costes humanos.
- **Matriz Comparativa:** Evaluación de alternativas según criterios de seguridad, coste y complejidad.
- **Mini-RAID:** Registro de Riesgos, Supuestos, Incidencias y Dependencias.
- **Generador de Resumen:** Exportación de resultados para informes finales.

## ⚠️ Privacidad y Seguridad
Esta aplicación funciona exclusivamente en el lado del cliente (navegador). No almacena ni envía datos a servidores externos. 
**No introduzca datos personales reales ni información confidencial.**

## 📖 Cómo usar
1. Ingrese los datos operativos en la sección de "Datos del Caso".
2. Revise los cálculos automáticos en la pestaña de resultados.
3. Complete la matriz comparativa y el RAID basándose en el análisis del caso.
4. Redacte la decisión final y exporte el resumen para su entrega.
